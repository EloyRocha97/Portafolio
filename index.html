Mis Proyectos

Api Dogs

El proyecto se desarrolló utilizando Express, React, Redux y Node. Se trata de una API que funciona como una página de perros, donde se pueden aplicar filtros de búsqueda, utilizar una barra de búsqueda y acceder a una sección con detalles de cada raza. Permite buscar y ordenar razas por nombre, peso y temperamento, y también ingresar el nombre directamente. La sección de detalles brinda información sobre origen, tamaño, esperanza de vida y comportamiento de la raza. Además, se puede agregar nuevas razas a través de un formulario. En resumen, esta API proporciona una experiencia interactiva y fácil de usar para explorar!

Pets-Bridge

El proyecto consiste en el desarrollo de una aplicación web utilizando Next.js en el front y Nest.js en el back, con el objetivo de crear una plataforma donde diferentes asociaciones y organizaciones dedicadas al rescate y cuidado de animales puedan publicar información sobre las mascotas que tienen disponibles para apción. El proyecto contiene una pasarela de pago(strapi), autenticacion de terceros (nextAuth), filtrados, dashboard de admin, subida de archivos a cloudinary!

E-comerce

El proyecto se desarrolló utilizando Express, React, Redux y Node. Se trata de un E-comerce de ropa basico. En el cual se puede logear con una cuenta, a traves de google o por gmail. Cuenta con una con un apartado inicial sobre la pagina, una seccion de tienda y otra para el contacto a la misma. La seccion de tienda cuenta con una barra de busqueda. Un paginado. Filtros segun el genero, los diferentes tipos de ropa y sus diferentes talles.

Proyecto Back

Este es un pequeño proyecto backend que desarrollé para una API con funcionalidades clave: solicitar turnos de un calendario y realizar compras de productos seleccionados. El principal objetivo de este proyecto es llevar a cabo diversos filtros desde el backend, debido a que hasta la fecha los realizaba desde la parte del frontend. Este enfoque proporciona una mayor flexibilidad y control sobre la manipulación de datos, mejorando la eficiencia y la seguridad de la aplicación. Tambien cuenta con un login y una pasarela de pago.

Back En Python

Este es mi primer proyecto de backend, desarrollado con Python y MongoDB. Este proyecto ofrece un CRUD básico que permite la gestión completa de usuarios: desde su creación hasta su búsqueda, edición y eliminación según las necesidades. Además, integra un sólido sistema de registro basado en JWT para garantizar la seguridad y la autenticación de los usuarios.
